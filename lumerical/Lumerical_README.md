# Lumerical Simulation Files

This directory contains the original Ansys Lumerical FDTD project files used for the electromagnetic simulations reported in the manuscript.

## Contents

Typical files may include:

```text
sensor_design.fsp
sensor_TE.fsp
sensor_TM.fsp
```

or equivalent project files used during the study.

## Requirements

- Ansys Lumerical FDTD Solutions
- Version 2024 R1 or newer recommended

## Included

- Device geometry
- Material assignments
- Source configuration
- Boundary conditions
- Monitors
- Simulation region settings

## Not Included

The following items are intentionally omitted while the manuscript is under peer review:

- Raw spectra
- MATLAB scripts
- Python scripts
- Figure generation codes
- Processed sensing datasets
- Supplementary analysis files

These materials may be released after publication.
