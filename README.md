# van der Waals Crystal Anisotropy Controls Dual-Channel Refractive Index Sensing in a TiO₂/α-MoO₃ Nanobar Metasurface

## Overview

This repository accompanies the manuscript:

> van der Waals Crystal Anisotropy Controls Dual-Channel Refractive Index Sensing in a TiO₂/α-MoO₃ Nanobar Metasurface

The work demonstrates that the biaxial anisotropy of α-MoO₃ enables asymmetric dual-polarization Fano resonances in a TiO₂ nanobar-pair metasurface, resulting in dual-channel refractive index sensing and polarization-dependent analyte fingerprinting.

## Key Results

| Metric | TE Channel | TM Channel |
|----------|------------|------------|
| Resonance Wavelength | 863.3 nm | 960.1 nm |
| Quality Factor | 87 | 31 |
| Sensitivity | 155.3 nm/RIU | 139.1 nm/RIU |
